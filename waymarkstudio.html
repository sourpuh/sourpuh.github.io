  1. 1. Install XIVLauncher and launch the game

    Open Dalamud settings using your preferred method, such as the chat command /xlsettings.

  3. 3. Navigate to Experimental Tab

    In the Dalamud settings window, click on the "Experimental" tab.

  4. 4. Add the Custom Plugin Repository

    Scroll down to the "Custom Plugin Repositories" section and paste https://puni.sh/api/repository/sourpuh into the last empty textbox.

  5. 5. Save Changes

    Press the '+' button right of the textbox, and then click the 'üíæ' (Save changes and close) button.

  6. 6. Open the Installer

    Open Dalamud plugin installer using your preferred method, such as the chat command /xlplugins and open the "All Plugins" section.

  7. 7. Install the Plugin

    Search for "Waymark Studio" in the plugin installer and proceed with the installation.

  8. 8. Open Waymark Studio

    Once installed, Waymark Studio can be opened using the chat command /wms or by opening the game's waymark window (default hotkey ctrl + m).
